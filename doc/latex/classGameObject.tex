Метод инициализации объекта на сцене. 

Добавляет объект на сцену, переданную в конструкторе.

\begin{DoxyWarning}{Предупреждения}
Перед инициализацией объекта на сцене убедитесь, что для объекта был вызван метод {\ttfamily void Q\+Graphics\+Pixmap\+Item\+::set\+Pixmap(const Q\+Pixmap \& pixmap)}. В противном случае метод вернет ошибку {\itshape You must to set a picture}. 
\end{DoxyWarning}


Переопределяется в \mbox{\hyperlink{classShieldDecorator_a57d1d95c4c46e86c9fc4afdfe6da83db}{Shield\+Decorator}} и \mbox{\hyperlink{classAbstractEnemyDecorator_acacd3bf0119d9609a87bdfa2ae25ca8d}{Abstract\+Enemy\+Decorator}}.
