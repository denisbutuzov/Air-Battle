void GameObject::init ( )
virtual

Метод инициализации объекта на сцене.

Добавляет объект на сцену, переданную в конструкторе.

Предупреждения
Перед инициализацией объекта на сцене убедитесь, что для объекта был вызван метод void QGraphicsPixmapItem::setPixmap(const QPixmap & pixmap). В противном случае метод вернет ошибку You must to set a picture.

Переопределяется в ShieldDecorator и AbstractEnemyDecorator.
